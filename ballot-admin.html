Setup guide

  1. Create a new Google Sheet
  2. Go to Extensions → Apps Script
  3. Replace all code with the contents of Code.gs
  4. Click Deploy → New deployment
  5. Type: Web app · Execute as: Me · Access: Anyone
  6. Copy the web app URL and paste it above
  7. Host entry-form.html on GitHub Pages or similar
  8. Paste the entry form URL above
